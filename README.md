# USSD Simulation System (Python + MySQL)
This project is a Python-based USSD simulation system that mimics basic mobile banking and telecom services using a command-line interface. It integrates a MySQL database to handle user registration, authentication, and account data management.

# Features
• User registration with account creation
•	Login with phone number and PIN authentication
•	USSD-style menu navigation system
•	Airtime purchase simulation with balance deduction
•	Account balance tracking and updates
•	Input validation and error handling

# Technologies Used
•	Python
•	MySQL (Database)
•	SQL queries (CRUD operations)

# Project Purpose
This project was built to simulate real-world USSD banking systems and strengthen my understanding of:
	•	Backend logic
	•	Database integration
	•	User authentication systems
	•	Command-line application design
  
# How It Works
•	Users register an account with personal details
•	Data is stored in a MySQL database
•	Users log in using phone number and PIN
•	The system provides a USSD-style menu for transactions
•	Transactions update the user’s balance in real time
